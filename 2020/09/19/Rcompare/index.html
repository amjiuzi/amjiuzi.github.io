T test

t-test 主要用于样本含量较小（例如n < 30），总体标准差σ未知的正态分布。t-test 是用t分布理论来推论差异发生的概率，从而比较在符合正态分布的样本中两个平均数的差异是否显著

t.test(x,y,mu,alternative,paired,var.equal)

paired = true, 对具有相同样本量的2组数据做对比;false(默认)对比非相同数目的2组数据
var.equal = false, x&y 不具有相同方差; true 具备相同方差

alternative two.sided less greater
不设置y 检验定值mu==x 检验定值mu < x 检验定值mu > x
设置y 检验y均值==x均值 检验y均值 < x均值 检验y均值 > x均值

单样本均值t检验

一鱼塘多年水中含氧量平均值为4.5 mg/L，现随机在该鱼塘采集10个样本，试检验该次抽样的均值是否与多年均值有显著差别

我们可以看到t.test函数给出的结果中包含了计算出的检验值(t= xxx)、自由度(df)、p值以及检验的原假设，由于p值大于 0.05, 否定原假设 ，则现状与之前没有显著性差异。通过现均值 4.421, 原均值 4.5 都落在95%置信区间 [4.23,4.61] 范围内，也可以证明无显著性差异

Anova

T检验只能分析两个类别的数据，比如性别（男、女），甲公司与乙公司，A组和B组之间数据的差异等
方差分析可分析三组及三组以上的数据，比如学历（本科以下、本科、本科以上）、专业（理科、工科、文科等）

基本步骤

  1. 箱线图查看各类别下数据分布情况
  2. 各类别下 指标值 正态分布检验
  3. 各类别下 指标值 方差齐性检验
  4. 各类别下 Kruskal-Wallis 检验
  5. 方差分析 aov()
  6. 多重T检验 pairwise.t.test()
  7. 多重wlicox检验 pairwise.wlicox.test()

  • 方差齐性检验
    方差比、Hartley检验、Bartlett检验都需要原始数据是正态分布，Levene检验和BF法对正态分布不是很依赖。比较常用的是Levene检验，适用于多组方差的比较，且对正态性没要求

  • 多重比较
    TukeyHSD 要求各组样本量一致, pairwise.t.test 可以容忍各组样本量不一致， 二者同属于 参数检验的范畴
    pairwise.wilcox.test 可以容忍各组样本量不一致，它属于非参数检验的范畴
