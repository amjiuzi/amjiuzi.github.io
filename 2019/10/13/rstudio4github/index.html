How to push/pull project to github by rstudio

1. Install R and rstudio

2. Install git and register it into Rstudio, get through them

  • 2.1 Install git
  • 2.2 Rstudio – tools – global options – git/SVN – git executable (xx/Git/bin/git.exe) – apply
  • 2.3 Rstudio – tools – global options – git/SVN – SSH RSA key – Create RSA key – ok – view public key – copy public keys
  • 2.4 Rstudio – tools – version control – Project Setup – git/SVN – git – ok

3. Signup a account in github

  • 3.1 Signup a github account
  • 3.2 Login github – settings – SSH and GPG keys – ssh keys – new ssh keys – paste keys from clipboard in step 2.3

4. Create a new project by rstudio and pull files from github

  • 4.1 Create new project
    rstudio – files – new project – new direction
  • 4.2 Pull files from github
    rstudio – git – pull(blue arrow with download direction)

5. Another way for 2.3 and 3.2 ( setting by bash.exe )

  • 5.1 Click xx/Git/bin/bash.exe and write code in it
  • 5.2 Coding in bash

5. Bring or create/edit project files in proj1 file

6. Select files u like to commit and write sm changing records in rstudio

7. Push files to github

  • 7.1 Push in rstudio
    rstudio – git – commit – push (green arrow with upload direction)

  • 7.2 Push in bash.exe
