Hexo Workflow

hexo clean

will clean up all database and files created formerly, what we can see is ./myblog/public folder will disappear

hexo generate

It create ./myblog/public/… folder according our markdown files under ./myblog/source/_post .
./myblog/public holds all materials being to display in gitee pages

hexo deploy

Deploy our web project to our web-space or local space

hexo server

Start local web server, we can browse https://localhost:4000 to check our page view. if we deploy to webspace, it’s unneccesary to run
