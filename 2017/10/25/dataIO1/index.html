2. 读取 csv/txt等本地文件 可以参考 read.csv/read.table/fread/scan/read.fwf(读取固定宽度数据)

3. 读取 xls,xlsx 可以参考 readxl/openxlsx

4. 读取 json文件, 可以参考 rjson

由于json数据中有很多NULL值字段,R在转换时会自动舍弃,所以想要保持原表规整方便后续合并需将 null 值替换为其他标签值如 NA
